Cased HTML Viewer — Build Pipeline

YAML is canonical. Everything else is a rendering frontend.

SOURCE (audit directory)
recon.yaml
findings.yaml
assets/*.svg
|
BUILD (node scripts/build-viewer.mjs)
Step 1: Bundle JS
rolldown bundles:
  rough.js (IIFE)
  rough-notation (IIFE)
  viewer.js (app logic)
→ single viewer-bundle.js
Step 2: Render HTML
Parse YAML → generate:
  narrative sections
  finding articles
  code evidence blocks
  remediation ledger
+ embed JSON data blob
Step 3: Assemble
Inline into template.html:
  <style> viewer.css
  @font-face base64 woff2
  <script> viewer-bundle.js
  <script type=json> data
  generated HTML content
|
OUTPUT (audit directory)
report.html
single file, zero external deps, ~300-500KB typical
OTHER FRONTENDS (same YAML source)
README.md (GitHub-native)
report.pdf (Typst, if on $PATH)
