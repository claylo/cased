Shape type mapping for the schema

type value Shape When to use
start Solid circle Entry point (auto-added as first step)
end Bull's-eye Terminus (auto-added as last step)
process Hollow circle Default. Code step that executes.
decision Hollow diamond Conditional branch, guard clause, match arm
input Parallelogram Data arriving: request, message, file read
store Cylinder DB access, cache, persistent write
ref Gray dot Reference to another flow / off-page connector
